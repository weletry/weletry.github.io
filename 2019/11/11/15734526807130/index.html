PProf性能分析

进行go性能分析及优化时，go语言提供了一个很有力的分析工具，那就是PProf。这个东西分析涉及：

  • runtime/pprof:采集程序（非常server）的运行数据进行分析
  • net/http/pprof：采集http server的运行时数据进行分析

可以获取到的信息

  1. CPU Profiling: CPU分析，可确定应用程序在主动消耗 CPU 周期时花费时间的位置
  2. Memory Profiling: 内存分析，在应用程序进行堆分配时记录堆栈跟踪，用于监视当前和历史内存使用情况，以及检查内存泄漏
  3. Block Profiling: 阻塞分析，记录goroutine阻塞等待同步的位置
  4. Mutex Profiling: 互斥锁分析，查看竞争的情况

CPU

$HOST/debug/pprof/profile，默认进行 30s 的 CPU Profiling，得到一个分析用的 profile 文件
